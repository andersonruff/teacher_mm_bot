# teacher_mm_bot
Esse projeto é uma idéia de um bot que auxili no aprendizado de um novo idioma, feito em python, sua função é enviar mensagem com perguntas em um outro idioma, essas perguntas
auxiliará a mentalizar e decorar frases, palavras e assim levando a um aprendizado realista, totalmente gratuito, foi feito com o auxilio de IA na correção do código e na cria-
ção do yml, eu entendo isso como uma automação de etapa, tudo pra poder fazer um projeto com mais eficiencia e agilidade, não precisamos inventar a roda se ela ja existe!!!
Tenho pretenção de evoluir o projeto pra um projeto de ensino mais consistente.
